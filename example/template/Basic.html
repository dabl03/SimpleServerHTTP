¿Porque hacer SimpleServerHTTP?

Como proyecto para portafolio, quise hacer una aplicación que emulara un banco. Entonces buscando API para actuar como servidor conseguí varias, pero, no son soportada por la pc que uso, asi que diseñe una basada en flask.

¿Como usar la biblioteca SimpleServerHTTP?

Para usar la biblioteca solo hay que heredad la clase UrlController y definir las funciones que envia la respuesta al cliente con el atributo UrlInfo("/Basi.html"). Tambien implementar una función para escuchar las peticiones.
